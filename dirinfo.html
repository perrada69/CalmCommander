.dirinfo

Recursively counts files, subdirectories and total size of any directory on your ZX Spectrum Next.

Description

.dirinfo is a standalone NextZXOS dot command that recursively scans a directory and reports:

  • Total number of files
  • Total number of subdirectories
  • Total size in bytes (read directly from FAT directory entries — no extra API calls)

Supports Long File Names (LFN), quoted paths with spaces, and up to 12 levels of directory nesting. Press BREAK to cancel a long scan at any time.

Usage

.dirinfo [path]
.dirinfo -s path
.dirinfo -h

If no path is given, the current directory (.) is scanned.

Options

-s path    explicitly set the target path
-h         show help

Examples

.dirinfo
.dirinfo games
.dirinfo -s "my games"
.dirinfo /nextzxos

Notes

Size shown is the sum of raw file sizes in bytes. Actual disk usage is higher — files are rounded up to cluster boundaries on FAT.
