<h1>Calm Commander</h1>

<p><strong>ZX Spectrum Next file manager</strong></p>

<p>
Calm Commander is a dual-panel file manager for ZX Spectrum Next, built for fast work with files, directories, disks, and runnable formats.
</p>

<p>
ZX Spectrum Next / NextZXOS / dual panel / mouse support / viewer plugins
</p>

<h2>Commander-style file work on the Next</h2>

<p>
Calm Commander brings a classic two-panel workflow to ZX Spectrum Next: left and right panels, keyboard-driven navigation, copying, moving, deleting, renaming, and directory management.
</p>

<p>
The project is still in development, but it already handles everyday file-management tasks, launching common file formats, and modular file viewing through external plugins.
</p>

<h2>Features</h2>

<ul>
  <li><strong>Two panels:</strong> left and right file panels for copying, moving, and comparing directories.</li>
  <li><strong>File operations:</strong> copy, move, delete, rename, create directories, and select multiple files.</li>
  <li><strong>Run files:</strong> supports common formats such as BAS, TAP, NEX, SNA, SNX, and Z80.</li>
  <li><strong>Search and select:</strong> find files by name and select or deselect matching entries.</li>
  <li><strong>File details:</strong> view size, date, time, attributes, and other details for the selected item.</li>
  <li><strong>Viewer plugins:</strong> text files, ZX screens, and other formats can be handled by external viewers.</li>
</ul>

<h2>Controls</h2>

<p>
Calm Commander is designed for quick keyboard use and also supports mouse interaction. The main actions are shown in the bottom command bar.
</p>

<ul>
  <li><strong>1</strong> - switch to left panel</li>
  <li><strong>2</strong> - switch to right panel</li>
  <li><strong>3</strong> - open viewer plugin</li>
  <li><strong>5</strong> - copy file</li>
  <li><strong>6</strong> - move file</li>
  <li><strong>7</strong> - create directory</li>
  <li><strong>8</strong> - delete selected item</li>
  <li><strong>0</strong> - open menu</li>
</ul>

<h2>Modular file viewers</h2>

<p>
The viewer system loads an external plugin based on the selected file type. The text viewer supports line and page scrolling; the ZX screen viewer displays 6912-byte Spectrum screens.
</p>
