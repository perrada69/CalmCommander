.del

Recursively deletes files and entire directory trees on your ZX Spectrum Next, with full LFN support.

Description

.del is a standalone NextZXOS dot command that deletes files and entire directory trees, including all subdirectories at any depth. Built on the same engine as the Calm Commander internal delete function.

  • Recursive deletion of entire directory trees
  • Full Long File Name (LFN) support — handles filenames with spaces and special characters
  • Quoted paths for names containing spaces
  • Runs at 28 MHz turbo for maximum speed
  • Press BREAK to cancel between files at any time

Usage

.del target
.del -s target

Options

-s path    explicitly set the target path
-h         show help

Examples

.del tmp
.del d:/backup
.del "old folder"
.del -s "d:/old backup"

Warning

Deletion is permanent. There is no recycle bin or undo on NextZXOS FAT filesystems.

Part of the Calm Commander project.
